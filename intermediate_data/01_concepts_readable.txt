================================================================================
ID: power_rule_integration
Difficulty: easy
Prerequisites: basic_algebra, exponents

Context:
Use this rule to integrate monomials of the form x^n. It is the most fundamental integration technique taught early in calculus.

Formula:
\int x^{n}\,dx = \frac{x^{n+1}}{n+1}+C \quad (n \neq -1)

Since \(\cos x\) is even, \(\int_{-\pi/2}^{\pi/2}\cos x\,dx = 2\int_{0}^{\pi/2}\cos x\,dx = 2\).


================================================================================
CONCEPT 54: Integrating \(|x|\) Over \([-a,a]\)
================================================================================
ID: integral_absolute_value_over_full_interval
Difficulty: easy
Prerequisites: absolute_value, definite_integrals

Context:
Because \(|x|\) is even, \(\int_{-a}^{a}|x|dx = 2\int_{0}^{a} x\,dx = a^2\).

Worked Example:
For \(a=3\), the integral equals \(9\).


================================================================================
CONCEPT 55: Finding the Constant of Integration Using a Given Point on the Antiderivative Curve
================================================================================
ID: determining_constant_of_integration_from_point
Difficulty: medium
Prerequisites: integration, solving_for_constants

Context:
Integrate to obtain a family of functions, then substitute the given point to solve for \(C\).

Worked Example:
Integrating \(3\cos x - 2\sin x\) gives \(3\sin x + 2\cos x + C\). Using \((\pi/2,1)\) yields \(C = -2\).


================================================================================
CONCEPT 56: Integrating \(\frac{\cos x}{\sin^2 x}\)
================================================================================
ID: integral_cos_over_sin_squared
Difficulty: easy
Prerequisites: basic_trig_integrals

Context:
Rewrite as \(\cot x\csc x\) or note that derivative of \(\csc x\) is \(-\csc x\cot x\).

Worked Example:
Result: \(-\csc x + C\).


================================================================================
CONCEPT 57: Integrating \(5 - 2^x\)
================================================================================
ID: integral_exponential_minus_power_of_two
Difficulty: easy
Prerequisites: exponential_integration

Context:
Integrate term‑wise: \(\int 5dx = 5x\) and \(\int 2^x dx = \frac{2^x}{\ln 2}\).

Worked Example:
Result: \(5x - \frac{2^x}{\ln 2}+C\).


================================================================================
CONCEPT 58: Integrating \(\frac{1}{(t-2)^2+5^2}\)
================================================================================
ID: integral_arctangent_quadratic_shift
Difficulty: easy
Prerequisites: inverse_trig_integrals

Context:
Use the standard form \(\int \frac{dx}{x^2+a^2}=\frac{1}{a}\tan^{-1}\frac{x}{a}+C\). Shift the variable accordingly.

Worked Example:
Result: \(\frac{1}{5}\tan^{-1}\left(\frac{t-2}{5}\right)+C\).


================================================================================
CONCEPT 59: Finding a Function \(f\) Satisfying \(\int e^x(f+f')dx = e^x\sin x\)
================================================================================
ID: identifying_f_from_integral_identity
Difficulty: medium
Prerequisites: integration_by_recognition, derivatives_of_trig_functions

Context:
Compare with the identity \(\int e^x(g+g')dx = e^x g\); thus \(f(x)=\sin x\).


================================================================================
CONCEPT 60: Integral of an Odd Function Over a Symmetric Interval Resulting in Zero
================================================================================
ID: integral_odd_product_over_even_denominator
Difficulty: easy
Prerequisites: function_symmetry

Context:
If the integrand is odd, the integral from \(-a\) to \(a\) vanishes.

Worked Example:
For \(\int_{-\pi/2}^{\pi/2}\frac{x\cos x}{1+\sin^2 x}\,dx\) the integrand is odd, so the value is 0.
